# Allyaria.Theming.Constants.Colors

`Colors` provides a consolidated library of named colors (CSS and Material), exposed as strongly-typed
`AllyariaColorValue` properties. It includes the full **CSS Web Color** set and the **Material Design** palette with
shade variants (50–900, A100–A700).
